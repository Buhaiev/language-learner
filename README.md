# language-learner
A website designed to help you learn a language. Translate words and add them to your vocabulary, then get tested and see which words you remember.

Explansion plans: 
- login and user authentication
- session review(remember how many words were revised each day)
- better, Anki-like word revision algorithm

To use the translation feature, provide your HuggingFace token as an environment variable HF_TOKEN.
